DNS Propagation Checker

Simply enter your websites domain name to see which IP address your domain is currently pointing at. We query a number of reliable public Domain Name Servers so you can find out whether your domain has fully propagated in various locations around the globe.

This app is free to use and open source, and has been made public facing to benefit other developers and website owners. No adverts, no cookies, just fast DNS lookup.

If you're interested in seeing how this was built, want to contribute or have any issues, take a look at the GitHub repo.
